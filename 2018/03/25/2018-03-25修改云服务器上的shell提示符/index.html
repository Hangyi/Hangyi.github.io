看着服务器命令行里的提示符灰蒙蒙的，不像我电脑上的oh-my-zsh的主题，色彩丰富。所以我就想能不能把腾讯云服务器里的shell里的提示符也改变一下颜色。

提示符是由一个环境变量定义的，叫做 PS1（是“prompt string one” 的简写）。我们可以通过 echo 命令来查看 PS1的内容。
